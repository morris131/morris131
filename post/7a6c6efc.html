back/java/zookeeper/zookeeper常用命令

发表于 2019-12-03

首先启动zookeeper服务端，然后通过zkCli.sh脚本启动客户端。

通过执行help命令，查询可用的zookeeper命令(随便输入一个zookeeper不存在的命令都会出现)。

ls

列出指定节点中下的所有子节点（不包含孙子节点）。

create

创建一个新的节点。

  • -e 创建临时节点
  • -s 创建顺序节点
  • acl 是针对这个节点创建一个权限的，如果创建权限了，则拥有权限的才可以访问

每个路径下的顺序阶段的编号都是从0开始自增，这个数字由父节点维护。

get

查看节点的内容

set

修改节点的内容

stat

查询节点的状态

  • cZxid：节点被创建的zxid
  • ctime：创建时间
  • mZxid：节点最后一次修改的zxid
  • mtime：修改时间
  • pZxid：子节点（或该节点）的最近一次创建/删除的时间戳对应（不含孙子节点）
  • cversion：子节点被修改的版本号
  • dataVersion：节点被修改的版本号
  • aclVersion：节点的ACL被修改的版本号
  • ephemeralOwner：若此节点为临时节点，值为该节点拥有者的会话id，否则为0
  • dataLength：数据长度
  • numChildren：子节点个数（不含孙子节点）

Zxid是一个64位的数字，它高32位是epoch用来标识leader关系是否改变，每次一个leader被选出来，它都会有一个 新的epoch。低32位是个递增计数。

delete

删除节点

deleteall

删除节点(支持递归删除)
